android studio快捷键

  • 添加书签：F3
    • 显示书签列表：cmd + F3
  • 隐藏其他窗口:control + w（自定义）
  • 定位到父类的方法：cmd + u
  • 在finder中打开文件：cmd + 点击tab
  • 查看方法参数列表:cmd + p
  • 快速查看方法定义:cmd + y
  • Esc：该操作仅仅把光标移回编辑器。
    Shift + Esc：该操作会关闭当前面板，然后把光标移回到编辑器。
  • 按住Alt竖排选中
  • 语句补全:cmd + shift + enter
  • 代码补全时enter不对后面修改tab对后面修改
  • cmd + alt + m 提取方法
  • contral + alt + h查看调用层级
